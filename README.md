#Generic Binary File Extractor Library
=====================================
##Scope
------
设计这个库用于处理从二进制文件中按照指定格式解析数据内容的过程。

##Target
-------

======================================================
* 使用C/C++编写
* 易于扩展：支持定制的文件格式
* 平台无关
    *文件系统存储
    *硬件（nandflash）存储
    *内存区间
* 支持Cache机制
    * LRU
    * MRU
    * FIFO
* 单元测试（Googletest）

=====================================================
